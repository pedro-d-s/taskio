(BR)
Este é um projeto que foi feito a fim de agilizar o gerenciamento de tarefas diário das pessoas. O nome desse gerenciador de tarefas é Taskio.

Com esse gerenciador, é possível criar tarefas em um determinado dia e alterar suas descrições. Além disso, é possível criar mais de 1, 2, 3 tarefas em um dia, se desejar. 

Esse gerenciador tem um design moderno e agradável aos olhos com cores laranja e azul marinho.

Com certeza esse projeto tem grandes oportunidades de uso e de agilizo de vida. Qualquer pessoa responsável usa um gerenciador de tarefas no seu dia a dia, não é mesmo?

(EN)
This is a project designed to streamline people's daily task management. The name of this task manager is Taskio.

With this manager, it's possible to create tasks on a given day and change their descriptions. Furthermore, it's possible to create more than 1, 2, or 3 tasks in a single day, if desired.

This manager has a modern and visually appealing design with orange and navy blue colors.

This project certainly offers great opportunities for use and streamlining daily life. Any responsible person uses a task manager in their daily routine, right?
